Space Alert Mission Generator for Android

Simply clicking the button on the main page will generate and start a mission using the current preferences. If you change the preferences make sure to create a new mission under the menu to get the new settings.

Menu Options

  • Type - Choose an already generated mission or to generate a random mission.
  • Reset - Resets the current mission to beginning.
  • New - generates a new mission with the current preferences.
  • Perferences - preferences to set for the mission.
  • About - Information about product.
  • Help - this page.
